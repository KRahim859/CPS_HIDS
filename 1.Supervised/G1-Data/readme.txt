
G1 instruction is used for controlling the movement. The G1 data has features namely G1, Z, F, E, X and Y features. 
The legitimate or benign instructions hence, labelled as Drop 'No'. 
The malicious instructions added programmatically to satisfy limits & thresolds set as per Rule based criteria and hence labelled as Drop 'Yes'. 
The file "G1_multiclass.csv" contains the multiclass dataset that contains the labels as Low, Medium and High.
This data can be used for first ML model that can learn set limits & thresholds in supervised manner.
